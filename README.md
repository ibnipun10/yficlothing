# yficlothing

Single-page static website for `yficlothing.in`.

## GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to the repository branch, usually `main`, and the root folder.
4. Keep the `CNAME` file as `yficlothing.in`.
5. In your domain DNS, point the domain to GitHub Pages.

## Instagram content

This static site links visitors to Instagram and WhatsApp. To show actual Instagram reels or posts on the page, add specific public post or reel URLs and embed them manually, or connect an approved Instagram API or widget later.
